# Relaxed JSON

```
pip install relaxedjson
```

See http://pythonhosted.org/relaxedjson/

A more relaxed JSON parser that supports missing quotes on keys.

### Relaxed JSON:

```python
>>> import relaxedjson
>>> relaxedjson.parse('{moose: "goose"}')
{'moose': 'goose'}
```
